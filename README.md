# Katalog Peralatan

Katalog personal untuk utilitas CLI, skrip rotasi proxy, dan sistem terdistribusi milik Hafizh Muzani.

## Stack
- **HTML5** — Single page static
- **TailwindCSS** — CDN via Tailwind Play
- **Vanilla JS** — Search & filter
- **Hallmark Anti-Slop Theme** — Atelier editorial layout

## Run Locally
```bash
python -m http.server 8000
# open http://localhost:8000
```
